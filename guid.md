2. ส่วนประมวลผล (Backend Logic)
เพื่อให้ GUI ทำงานได้เหมือนกับ Model ที่เราทำไว้ ต้องมีขั้นตอนดังนี้:

Data Preprocessing: นำค่าที่ Input เข้ามาทำการ Scaling โดยใช้พารามิเตอร์ mean และ std จาก StandardScaler ตัวเดิมที่คุณใช้ใน Notebook (แนะนำให้ Save scaler เป็นไฟล์ .pkl)

Clustering Prediction: นำข้อมูลที่ Scale แล้วไปหา "ระยะห่างที่ใกล้ที่สุด" (Euclidean Distance) กับ Centroids ของทั้ง 5 Clusters ที่เราหาได้

Mapping: เมื่อได้ Cluster ID แล้ว ให้ดึงข้อมูล Persona และ Recommendation ที่เตรียมไว้มาแสดง

3. ส่วนแสดงผลและการแนะนำ (Output & Recommendation)
เมื่อผู้ใช้กดปุ่ม "วิเคราะห์ตัวตน" ให้แสดงผลตาม Cluster ดังนี้:

กรณีเป็น Cluster 0: "The Leisure-First Dreamer"
Persona: รักอิสระ ให้ความสำคัญกับความสุขส่วนตัว แต่การเงินยังไม่มั่นคง

สินค้าแนะนำ: บริการสตรีมมิ่ง, ท่องเที่ยวแนวประหยัด, สินค้าแฟชั่นราคาเข้าถึงง่าย

การตลาด: "เติมเต็มความสุขวันนี้ โดยไม่ต้องรอให้รวย" (ใช้โทนใจดี เป็นกันเอง)

กรณีเป็น Cluster 1: "The Vulnerable Spendthrift"
Persona: สายสังคมตัวจริง ชอบทำกิจกรรม แต่มีภาระการเงินสูงหรือเงินออมน้อย

สินค้าแนะนำ: สินค้าผ่อน 0%, บริการรับประทานอาหาร/บัตรเครดิตสะสมแต้ม, ประกันภัยเบี้ยต่ำ

การตลาด: "สนุกกับชีวิตได้เต็มที่ พร้อมตัวช่วยจัดการทุกค่าใช้จ่าย" (เน้นความคุ้มค่าและความสะดวก)

กรณีเป็น Cluster 2: "The Financially Secure Elite"
Persona: เศรษฐีเงียบ มีรากฐานการเงินแข็งแรงมาก แต่ไม่เน้นโอ้อวด

สินค้าแนะนำ: กองทุนรวม/การลงทุนระยะยาว, สินค้าสุขภาพพรีเมียม, Quiet Luxury Brands

การตลาด: "คุณค่าที่ยั่งยืน สำหรับผู้ที่เลือกสิ่งที่ดีที่สุดให้ตัวเอง" (ใช้โทนเรียบหรู น่าเชื่อถือ)

กรณีเป็น Cluster 3: "The Hard-Working Hustler"
Persona: นักสู้เพื่อความสำเร็จ งานต้องมาก่อน และชอบเข้าสังคมเพื่อคอนเนกชัน

สินค้าแนะนำ: Gadget รุ่นท็อป, คอร์สสัมมนาธุรกิจ, อาหารเสริมบำรุงสมอง, บริการ Concierge

การตลาด: "เครื่องมือสู่ความสำเร็จ สำหรับคนที่ไม่เคยหยุดพัฒนา" (ใช้โทนปลุกใจ ทะเยอทะยาน)

กรณีเป็น Cluster 4: "The Aspirational Underdog"
Persona: มีความทะเยอทะยานสูง แคร์ภาพลักษณ์และสถานะสังคมมากกว่ารายได้จริง

สินค้าแนะนำ: สินค้าแบรนด์เนมรุ่นเริ่มต้น (Entry-luxury), คอร์สพัฒนาบุคลิกภาพ, สินค้าที่เป็นกระแส

การตลาด: "ก้าวสู่ตัวตนที่เหนือกว่า ยกระดับชีวิตคุณในทุกย่างก้าว" (ใช้โทนสร้างแรงบันดาลใจ)
